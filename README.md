# Word Collector

Word Collector is a simple android, that allows you to make your own dictionary by collection word and their definitions.

Word Collector is the first android app that i have build, with it i wanted to put into practice some of the things that i have learned 
about building android app.

With this app i practice about:
    
*		Kotlin
*		Unit Testing
*		Activities
*		Fragments
*		Navigation Component	
*		ViewModel
*		LiveData
*		Room Database
*		MVVM


## How to use

The app has basic usage you can write a word and a definition that you would like to save, you can separate the word into different categories.
The app also let you quickly look a word up in Google Translate with out leaving the app.  

## How to run the project

You can clone the repository and run the project in Android Studio.
OR
[You can download the app from the play store](https://play.google.com/store/apps/details?id=com.yllxh.wordcollector)
